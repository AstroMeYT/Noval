# Noval Knowledge Base Editor

Noval is a knowledge base editor based on my WebNote/Nexus project. It allows the user to use a custom rich text format, along with linking to other pages, a map view of your pages, and many more features.

## How to Use

To use Noval in the web, you can go to [this page](https://AstroMeYT.github.io/Noval). It is also able to be executed by compiling it with Electron, or you can use my custom [Muffin Package Manager](https://github.com/AstroMeYT/muffin) project to execute this as a native Gnome app for Linux.

## Features

- Rich text formatting for clear and concise notes.
- Ability to link pages inside of a page for ease of connecting related notes.
- AES-GCM encryption with PBKDF2 for password protection of your whole Noval database.
- Folder management of your files
- A Web graph view of of all of your notes and others that they link to.
- Import and export your databases with ease.
- More to come!

## Contribution

If you wish to make changes to the code, feel free to do so! Create a pull request after you have verified these check:

- The code compiles successfully without any visual or functional errors.
- If the code makes changes to the UI, please make sure it follows the same design as the current version.
- The code will not inflict with the privacy of users' data whatsoever. This includes sending data to servers or 3rd parties, collection of user data, or breaking the local encryption process.
