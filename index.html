    How to Use Noval

    • Write your notes naturally. Highlight text and use the toolbar to change Font, Size, and Color.
    • Switch easily between Edit Mode and View Mode using the dropdown in the toolbar.
    • Link concepts together using double brackets: [[Ideas]]. Select it or close the brackets to create a link instantly!
    • Switch to the Graph tab to explore your spiderweb map of connections.

    About Noval

    Noval is an incredibly powerful, premium rich-text knowledge base running entirely inside your local web browser using a custom JSON Abstract Syntax Tree (AST) backend.

    By combining advanced features like interactive mind-mapping, automatic bi-directional links, dynamic table of contents sidebar rendering, and secure on-device cryptographic locking, Noval gives you complete ownership over your intellectual thoughts.

    All data remains completely on your local machine within IndexedDB. There are no tracking scripts, telemetry channels, or remote database synchronization. Your data belongs exclusively to you.
